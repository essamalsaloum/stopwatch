# Stopwatch

A simple stopwatch app.

## Features
- Display time
- Start timer
- Stop timer
- Clear timer
- Save time (laps)

## Built With
- ES6 (Transpiled by Babel)
- React
- Redux
- Webpack 2
- Jest (Testing)

## Install Dependencies
run `npm install`

## Launch Project
`npm run dev` and navigate to `http://localhost:3000/webpack-dev-server/`

## Running Tests
`npm run test`
